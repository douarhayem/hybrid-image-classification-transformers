# hybrid-image-classification-transformers
 CNN–Transformer Fusion for Image Classification

This project compares different deep learning models for image classification.

Models:
- EfficientNet
- Swin Transformer
- Early Fusion
- Mid Fusion
- Late Fusion

Datasets:
- CASIA 2.0
- EuroSAT

Framework:
- PyTorch
- Google Colab
